# 🎬 Movie Library (Личная кинотека)

Сабирзянов Марсель 

## Описание

Movie Library — это десктопное приложение для хранения и управления коллекцией фильмов. Программа позволяет добавлять, удалять, фильтровать и просматривать фильмы с сохранением всех данных в JSON-файл.

## Функциональные возможности

✅ Добавление фильмов с указанием названия, жанра, года и рейтинга  
✅ Валидация ввода (год — число от 1888, рейтинг — от 0 до 10)  
✅ Фильтрация по жанру и году выпуска  
✅ Удаление выбранных фильмов  
✅ Автоматическое сохранение в `movies.json`  
✅ Сортировка фильмов по названию  

## Установка и запуск

### Требования
- Python 3.8 или выше
- Tkinter (встроен в Python)

### Запуск
```bash
python movie_library.py
