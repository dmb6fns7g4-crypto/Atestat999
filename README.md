📜 Random Quote Generator
Сабирзянов Марсель Random
Quote Generator — это графическое приложение на Python с использованием Tkinter, которое позволяет:

Генерировать случайные цитаты из предустановленного списка
Фильтровать цитаты по автору и теме
Добавлять новые цитаты в коллекцию
Просматривать историю сгенерированных цитат
Сохранять историю и коллекцию цитат в JSON-файлы
Функциональные возможности
✅ Генерация случайной цитаты
✅ Фильтрация по автору и теме
✅ История последних цитат (до 50 записей)
✅ Добавление новых цитат
✅ Сохранение данных в JSON
✅ Валидация ввода при добавлении цитат

Установка и запуск
Убедитесь, что у вас установлен Python 3.8+
Скачайте файл quote_generator.py
Запустите программу:
python quote_generator.py
